Bootstrap Features


In my Online CV, I used several Bootstrap features. In this Page, I'll break down the Bootstrap components and classes I used throughout the CA.

In my Navbar, I used .navbar as my main navigation container, navbar-expand-lg which is I used for responsive navigation for larger screens, navbar-light bg-light which is a light background for the navbar, navbar-toggler which is the toggler button for collapsing and expanding the navigation, navbar-brand which is a logo inside the navbar. navbar-nav: Container for the navigation links, nav-item which is a Individual navigation item. nav-link which I used for the style of navigation links. The fixed-top class, keeps the navbar at the top of the page regardless of scrolling, ensuring a consistent and user-friendly experience. For my layout, I used containers which helped structure my code appropriately to look user-friendly. I used container mt-5 which adds a top margin to the container. I used buttons btn for basic button styling and btn-primary for the button colour.

For my Images, I used img-fluid for responsive images that scale with its parent container and rounded-circle which rounds the image into a circle. For my heading, I used heading tags with bootstrap styling for h1, h2, h3.I used lead for a larger font size for introducing text. I used Bootstrap's grid system when placing elements within containers, creating a responsive layout.

I used Collapse, data-bs-toggle=collapse which activates the collapse functionality, data-bs-target specifies the target element to be collapsed, and a collapse class that hid the content initially. To display my contact information I used modal. I used modal-dialog which wraps around the actual content. The modal-header is where I put a title to let users know what the popup is about. The main body of the popup, where I present information or features, is called the modal-body. I added Bootstrap's list group feature which helped me organise information cleanly and stylishly. The list-group class helped me create well-structured lists, and list-group-item ensures each item looks neat. When it comes to spacing, the mt-5 class came in handy, adding a comfortable margin at the top to improve the visual appeal. Lastly, I used the script to link to Bootstrap's JavaScript for enabling Bootstrap features.
